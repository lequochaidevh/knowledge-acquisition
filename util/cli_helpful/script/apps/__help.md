# Guilde:

### Get help
```sh
adddocshelp --help
catcmdhelp --help
batcmdhelp --help
```

### Add new document `<file name>` to `<module>`.
```sh
adddocshelp <file name> <module>
```

### Check `<module>` list
```sh
catcmdhelp --list
```

### Get command helpful in documents with `<module>`
```sh
catcmdhelp <module>
# or:
batcmdhelp <module>
```

### All applications were installed at: `[path]`
```sh
# read path info
cat  /usr/local/bin/docs_dir.txt
```
